# Case Studies

All four projects use a single environment. To install the necessary dependencies, run:  

```bash
pip install -r requirements.txt
```

## Running the Servers  

- **SMS Spam Classifier**  
  Run the server using:  
  ```bash
  python spam_words.py
  ```

- **Summarization**  
  Run the server using:  
  ```bash
  python summary.py
  ```

- **Text Classification**  
  Run the server using:  
  ```bash
  python test_model.py
  ```

- **Sentiment Analysis**  
  Run the server using:  
  ```bash
  python sentiment.py
  ```
